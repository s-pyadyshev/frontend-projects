html-boilerplate

Gulp, Pug, Less, Animate.css, UIkit 3.2.1


Project build

$ cd html-boilerplate

$ npm i

$ npm i gulp -g

$ gulp build


UIkit custom icon generation

$ cd uikit-develop

$ yarn install

Add new icons to uikit-develop\src\images\icons

$ yarn compile
